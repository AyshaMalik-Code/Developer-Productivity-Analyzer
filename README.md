# 🚀 Developer Productivity Analyzer

A Python Streamlit application that analyzes GitHub developer productivity and repository statistics.

## Features
- GitHub profile analytics
- Followers and repository statistics
- Total stars calculation
- Programming language distribution
- Top repository detection
- Interactive dashboard

## Tech Stack
- Python
- Streamlit
- GitHub API
- Pandas
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
